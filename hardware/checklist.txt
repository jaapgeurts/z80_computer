Checklist before fabrication

1. Check all unused/free ports are added added to headers
2. Check all unused inputs are not floating
3. Check reset button can be bridged by jumper
4. Check critical traces can be cut for installing bodges
5. check zilog datasheets for direction of pins
6. double check wiring on pins of all ICs (following datasheets). Check pins too
7. check clock freq on PSG
8. PS/2 buffer + PS/2 host->dev communication - COMPLETE
9. Add port buffers where necessary
10. Check all wiring lightup in kicad
11. Add 5v input plug; bypassing regulator
12. Test points?
13. check each chip has 100n bypass cap
14. lcd displays: check which side is top. (orientation)
15. add led for halt 
16. power led
17. label ports
18. add expansion port 
19. add jumpers for clock speed
20. NMI button
21. 3 user leds (r g b)
22. 3 user buttons
23. show switch settings on silkscreen
24. show battery type on silkscreen
25. show port pinouts on silkscreen
26. check pcb power switch if it matches footprint
27. check ic pincounts on pcb
